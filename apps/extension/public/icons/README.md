# Icons

Please add the following icon files in this directory:

- `icon-16.png` (16x16)
- `icon-32.png` (32x32)
- `icon-48.png` (48x48)
- `icon-128.png` (128x128)

You can create simple placeholder icons using any image editor or online tools like:
- https://www.favicon-generator.org/
- https://www.canva.com/

For development, you can temporarily use any PNG files with the correct dimensions.
